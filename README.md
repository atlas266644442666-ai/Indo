# Туристический сайт — Путешествия по Индонезии
Простой статический сайт с информацией о турах и достопримечательностях Индонезии.

## Структура
- `index.html` — главная страница
- `tours.html` — список туров
- `about.html` — страница о компании
- `styles.css` — стили оформления
- `images/` — папка для изображений (опционально)
